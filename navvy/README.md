# Navvy

Diagnose a codebase through its git history before reading any code. Runs five git commands that surface churn hotspots, bus factor risks, bug clusters, velocity trends, and firefighting patterns, then synthesizes a diagnostic report.

Based on [Git Commands You Should Run Before Reading Code](https://piechowski.io/post/git-commands-before-reading-code/) by Adam Piechowski.

## Usage

```
/navvy
```

Invoke directly when you want to orient yourself in an unfamiliar codebase or assess the health of a project you're about to work on.
