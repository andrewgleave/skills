# 5-steps

Agent skill using Elon Musk's 5-Step process.

Challenges requirements, removes unnecessary work, and simplifies design and requirements before writing code. Works through: challenge requirements, delete the unnecessary, simplify the design, accelerate iteration, then build. Helps to catch the most expensive mistakes before a line is written.

Activates on build/implement/automate/migrate requests. Does not activate for bug fixes, simple edits, or general code writing.
